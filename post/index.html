Projects

Project 1 : Gemstone Price Estimator

Created a tool that estimates the gemstone prices to help the gemstone buyers when they want to buy. Imported the data to python environment and did EDA ,feature engineering ,feature selection, etc. Random Forest Regressor is used for the prediction. Built an API using flask. Link to GitHub repository

Project 2: Customer Segmentation

Created a tool that divide the online retail customers in to different groups based on the segmentation the business people can understand which segment they should target. Imported the dataset to python environment, Dataset preparation from existing data, Data engineering, Feature Selection etc. Kmeans clustering is used for this customer segmentation. Elbow method is used to get optimal K value and silhouette score is used to evaluate the model. Built an API using Flask.

Project 3: Image Classification On Balls Dataset

I built a project called ball classifier to identify balls from different sports. This could be useful for someone who is new to sports from certain country. They could take the picture of the ball and the app could which ball it is .I was able to get the model to predict more than 90% accuracy after minimal tunning .To get these results i used the CNN(convolutional neural network).
