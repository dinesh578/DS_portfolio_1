Project 2: Customer Segmentation

  • Created a tool that divide the online retail customers in to different groups based on the segmentation the business people can understand which segment they should target.
  • Imported the dataset to python environment, Dataset preparation from existing data, Data engineering, Feature Selection etc.
  • Kmeans clustering is used for this customer segmentation.
  • Elbow method is used to get optimal K value and silhouette score is used to evaluate the model.
  • Built an API using Flask.
